Description

Currently the project is still in a prototyping phase, hence the game not having a real name yet.

The game so far is focused on first person shooter where you fight various NPC in an enclosed enviroment.

Some Features and mechanics so far
  • Double jump
  • Enemies AI that prioritise different actions
    • attacking the player
    • collecting power ups
    • and attacking the enviroment
  • Enemy power ups
    • the power of a single enemy can change through out the level
